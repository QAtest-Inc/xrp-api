RippleAPIError

A wrapper for one or more error messages that occurred when processing a request.

  • Type: Object

This type can contain the following fields:

Field Type Required? Description
message String Required A human-readable error message summarizing the problem(s) that occurred.
errors Array of ErrorItem Required A list of errors that occurred when processing this request.
